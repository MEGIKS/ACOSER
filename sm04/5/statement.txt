Напишите функцию:
void to_half_float(int value, unsigned short *hf);
Функция преобразовывает целое число value в вещественное число половинной точности
(half-precision floating point), сохраняемое по указателю hf.
В вещественном числе половинной точности вещественное число представляется 16-битным значением.
Знак занимает старший бит, порядок - 5 бит, мантисса - 10 бит. Порядок кодируется со сдвигом (bias) 15,
старший бит мантиссы не хранится. Порядок, равный 0, используется для представления 0 и
денормализованных чисел, порядок 31 используется для NaN и INF.
Функция to_half_float выполняет преобразование целого знакового числа в вещественное число
половинной точности с математическим округлением к ближайшему представимому целому модуля числа. Если целое
число по модулю слишком велико, вырабатывается значение INF соответствующего знака.
Примеры:

0 -> 0
1 -> 0x3c00
-1 -> 0xbc00
1024 -> 0x6400
32758 -> 0x77ff
65536 -> 0x7c00
-120000 -> 0xfc00
65519 -> 0x7bff
