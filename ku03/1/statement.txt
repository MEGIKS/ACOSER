Напишите подпрограмму process, которая вычисляет выражение

A * X * X - 5 * B * X + 2 * C

числа A, B, C — это переменные, определенные вами в вашей единице компиляции,
X передается в регистре %edx, и результат возвращается в регистре %edx.
Все входные переменные и результат — 32-битные беззнаковые числа.
Все вычисления ведутся с 32-битными беззнаковыми числами, переполнение игнорируется
(то есть в качестве результата всегда берутся младшие 32 бита).
Соблюдайте правила использования регистров в
стандартном соглашении о вызовах.
Например, при A == 1, B == 2, C == 3, X == 4 результат вычислений равен 4294967278.
