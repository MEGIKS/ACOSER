На языке ассемблера напишите подпрограмму tree_find
со следующим прототипом (на Си):
int tree_find(struct Node *root, double key, char **pstr);
Узел бинарного дерева поиска определен следующим образом:
struct Node
{
    double       key;
    struct Node *left;
    struct Node *right;
    char        *str;
};
Подпрограмма tree_find возвращает 0, если ключ поиска key
не найден в дереве, и 1, если ключ найден. Если ключ найден, по указателю
pstr сохраняется указатель str из соответствующего
узла дерева. Если ключ не найден, значение по указателю pstr
не изменяется. Если pstr — нулевой указатель, выполняется
просто поиск в дереве без сохранения поля str.
При сравнении на равенство используйте допуск 1e-8.
Все ключи в дереве уникальные.
Подпрограмма должна соблюдать стандартные соглашения о вызовах.
Должен использоваться стандартный пролог и эпилог функции.
Для вычислений с плавающей точкой используйте SSE.
